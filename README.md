# 🔥 Fire Solutions Limited – Fire Safety & Protection Platform

> **Protecting Lives • Securing Assets • Building Safer Communities**

## 📖 About The Project

**Fire Solutions Limited** is a modern web platform dedicated to promoting fire safety awareness, emergency preparedness, and professional fire protection services. The platform serves as a comprehensive resource for individuals, businesses, and organizations seeking reliable fire prevention guidance, safety solutions, and emergency response information.

Designed with accessibility and usability in mind, the website delivers critical fire safety knowledge through an intuitive and responsive user experience.

## 🚀 Installation & Setup

### Clone the Repository

```bash
git clone https://github.com/your-username/fire-solutions-limited.git
```

### Navigate to Project Directory

```bash
cd fire-solutions-limited
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm start
```
